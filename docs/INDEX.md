## 文档使用建议

### 新用户
1. 从 [README.md](../README.md) 开始了解项目
2. 阅读 [QUICK_START.md](../QUICK_START.md) 快速上手
3. 根据需要查阅 [使用指南](#-使用指南-guides)

### 开发者
1. 参考 [Backend目录结构](../backend/docs/BACKEND_STRUCTURE.md) 了解代码组织
2. 查阅 [API实现总结](../backend/docs/API_IMPLEMENTATION_SUMMARY.md) 了解API设计
3. 遵循 [文档管理规范](../DOCUMENTATION_GUIDELINES.md) 编写文档

### 问题排查
1. 查看 [问题修复记录](#-问题修复记录-fixes) 寻找类似问题
2. 参考 [使用指南](#-使用指南-guides) 中的常见问题
3. 查阅 [API测试指南](./guides/API_TEST_GUIDE.md) 进行调试
